# Git으로 관리하는 파일의 상태
1. untracked
   - 추적안됨
      : 한번도 커밋되지 않은 파일


2. tracked 
   - 스테이지됨(staged)
        : 커밋할 파일로 선택해놓은 파일. add명령어
   - 수정없음(unmodified)
        : commit명렁어
   - 수정함
        : commit 이후 파일을 수정한 상태.
